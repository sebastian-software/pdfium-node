# pdfium-node-linux-x64-gnu

Linux x64 glibc native package for `pdfium-node`.

This package contains the Linux x64 glibc native PDFium addon for `pdfium-node`.

It supports selected-page PNG and JPEG rendering.

## Platform Support

This package is part of the stable `pdfium-node` support matrix for Linux x64 glibc on active Node.js LTS releases.
